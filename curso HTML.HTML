Explorando Tags HTML

HTML (Hypertext Markup Language) é a linguagem padrão para criar páginas da web. Ela utiliza "tags" para estruturar e organizar o conteúdo. Vamos explorar algumas das tags mais importantes:

    • História do HTML

      1989 - Proposição do Conceito de WWW por Tim Berners-Lee:

      O cientista da computação britânico Tim Berners-Lee propôs o conceito de um sistema de gerenciamento de informações
      baseado em hipertexto enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear).

      1990 - Criação do HTML:

      Em 1990, Tim Berners-Lee desenvolveu o primeiro conjunto de especificações para o HTML enquanto trabalhava no CERN.
      Nesse ponto inicial, o HTML era muito simples, com apenas cerca de 18 elementos.

      1993 - Lançamento do HTML 2.0:

      A primeira especificação formal do HTML foi lançada como HTML 2.0. Essa versão incluía a introdução de formulários, scripts e tabelas.

      1995 - HTML 3.0 e HTML+:

      O HTML 3.0 introduziu tabelas mais complexas e suporte para applets. No entanto, muitos recursos propostos não foram implementados.
      Ao mesmo tempo, surgiram variantes como HTML+ que incluíam mais funcionalidades.

      1997 - HTML 4.0:

      O HTML 4.0 foi um grande avanço e trouxe consigo recursos como CSS (Cascading Style Sheets),
      suporte para scripts (JavaScript) e a introdução de formulários mais avançados.

      1999 - XHTML 1.0:

      O XHTML (Extensible Hypertext Markup Language) 1.0 foi introduzido como uma reformulação do HTML 4.0 em XML (eXtensible Markup Language).
      O XHTML foi projetado para ser mais limpo e mais compatível com as regras do XML.

      2008 - HTML5:

      O desenvolvimento do HTML5 começou formalmente em 2004, liderado principalmente pela WHATWG (Web Hypertext Application Technology Working Group).
      Em 2008, a W3C (World Wide Web Consortium) também começou a trabalhar no HTML5.
      Essa versão trouxe uma série de recursos novos e avançados, incluindo suporte nativo para áudio e vídeo,
      elementos de semântica mais rica, API de armazenamento local, e muito mais.

      2014 - HTML Living Standard:

      Em vez de lançar novas versões, o HTML passou a ser tratado como um "Padrão Vivo" (Living Standard) pela WHATWG, onde as especificações são continuamente atualizadas para refletir a evolução da web. Ao longo dos anos, o HTML evoluiu para se tornar uma linguagem robusta, oferecendo suporte a uma ampla gama de funcionalidades para criar conteúdo na web de maneira eficaz. Sua evolução continua à medida que novas tecnologias e requisitos emergem na paisagem da web.

    Tags

    <!DOCTYPE html>

    Esta declaração define a versão do HTML que está sendo utilizada (HTML5, neste caso).

    <html>

    A tag <html> é a raiz do documento HTML. Ela engloba todo o conteúdo da página.

    <head>

    O elemento <head> contém metadados, como o título da página, links para estilos e scripts, entre outros.

    <title>

    A tag <title> define o título da página, que é exibido na barra de título do navegador.

    <body>

    O elemento <body> contém todo o conteúdo visível da página, como textos, imagens e links.

    <h1>, <h2>, <h3>, ..., <h6>

    Essas tags são usadas para definir cabeçalhos de diferentes níveis. <h1> é o mais importante e <h6> é o menos importante.

    <p>

    A tag <p> é utilizada para criar parágrafos de texto.

    <a>

    A tag <a> cria links. O atributo href especifica o URL de destino.

    <img>

    A tag <img> insere imagens na página. O atributo src define o caminho da imagem.

    <ul>, <ol>, <li>

    Essas tags são usadas para criar listas não ordenadas (<ul>), ordenadas (<ol>), e itens de lista (<li>).

    <div>

    A tag <div> é um contêiner genérico que é frequentemente usado para agrupar e estruturar outros elementos.
